# Introdução

Desafio Node + Nginx + Mysql + Docker FullCycle - Aplicação Node com proxy reverso no Nginx + Mysql

# Execução

Dar pull no projeto:

- git pull git@github.com:LuizHonorato/node-nginx-mysql-docker-challenge.git

Buildar a aplicação

- docker-compose up -d --build
